# VINKL — Session Log 2026-02-23

## Project Overview

**Brand:** VINKL — European architectural furniture brand (formerly WINKL)
**Product:** Adjustable teak corner wall shelf (85°–95°), 299 €
**Stack:** Next.js 16 App Router, TypeScript, Tailwind CSS v4, Headless Shopify, next-intl (DE/EN)

- Scalable folder structure: `src/` with `app/`, `components/`, `lib/`, `types/`, `config/`, `data/`, `messages/`

## Key Architectural Decisions

1. **Mapper pattern** — Shopify raw types never leak outside `lib/shopify/`. All consumers use app-level `Product` type.
2. **FLAGSHIP_SLUGS** — `Set<string>` determines which products get editorial treatment vs generic detail.
3. **`resolveProduct()` fallback chain** — Shopify → static data → null. Enables development without Shopify.
4. **Absolute-positioned hero image** — Desktop: right 42% panel. Mobile: full background at 12% opacity.
5. **Tailwind v4 `@theme inline`** — All design tokens (colors, spacing, motion) defined in CSS, not JS config.
6. **WebP image pipeline** — 5 product images optimized from 14.6 MB → 293 KB total. Sharp-based conversion.
7. **Bilingual alt texts** — `images` translation namespace ensures all alt texts are localized DE/EN.
8. **Zero-dependency gallery** — Product gallery with zoom, swipe, thumbnails — no external libraries.
9. **Repeating asymmetric grid** — About page uses consistent 5/7 label–heading–body rhythm.
10. **Editorial index layout** — Journal uses numbered entries with horizontal rules, not cards.
